Discord Duplicati Notifications

James Lloyd

Nov 7, 2022 1 min

Been trying to get my duplicati backup notifications into discord for a while, there are some bash scripts that i managed to get working, however, I wanted something a little more informative.  

So I wrote a little script to utilise the send-http-url report advanced option. To give notifications that look like the below

https://duplicati-notifications.lloyd.ws will generate a URL to forward your backup report to a discord webhook.  Using the backup name and webhook URL to generate the reporting send-http-url, you can then add that generated URL to your backup job in the advanced options (see below). Pick send-http-url from the Add advanced option dropdown, add the provided URL and save your backup job
